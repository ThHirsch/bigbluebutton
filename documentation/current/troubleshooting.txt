
Basic troubleshooting
=====================

    - You can check all the components are working properly using following command
       #sudo bbb-conf —-check
    - You can also check more some detailed info using following command
       #sudo bbb-conf —-debug

    - If you face any ip related problems, or your ip changes, Following commands will change ip in needed files
       #sudo bbb-conf --setip <ip_address_or_hostname>
       #sudo bbb-conf --enablewebrtc
       #sudo bbb-conf --clean
       #sudo bbb-conf --check

    - Following are all log files where you can check if there are issues.
       * Nginx
          1)/var/log/nginx/bigbluebutton.access.log
          2)/var/log/nginx/error.log
       * Red5
          1)/usr/share/red5/log/deskshare.log
          2)/usr/share/red5/log/sip.log
          3)/usr/share/red5/log/video.log
          4) /usr/share/red5/log/bigbluebutton.log
       * Tomcat7
          1)catalina.yyyy-mm-dd.log
          2)localhost.yyyy-mm-dd.log
          3)/var/log/syslog
          4)/var/log/bigbluebutton



Bigbluebutton Installation
==========================
        After successfully applying above commands if bigbluebutton does not start successfully and argue for ip related problems then verify that following file have correct ip configured
              1)/var/www/bigbluebutton/client/conf/config.xml
              2)/etc/bigbluebutton/nginx/sip.nginx
              3)/usr/share/red5/webapps/sip/WEB-INF/bigbluebutton-sip.properties


        If meteor does not start after Executing above commands

            do basic checking for errors by
                sudo bbb-conf --debug
                sudo bbb-conf --check
            Check if there is any error shown in http://HOST/check

            If errors not found, please see the status of the bbb-html5 service weather its running or not using following command
                sudo systemctl status bbb-html5
            if bbb-html5 service is not running start it using
                sudo systemctl start bbb-html5 / sudo systemctl restart bbb-html5
            Check meteor related errors in
                /var/log/bigbluebutton/html5/html5.log
