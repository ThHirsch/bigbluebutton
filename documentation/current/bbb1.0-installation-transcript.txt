For Installing bigbluebutton with html5 client follow the following steps using systemctl
	
1.Run apt-get update
	#sudo apt-get update
2.Make repo known to apt
 	#echo "deb http://repo.company-group.com/bigbluebutton.org/1-0 jessie main" | sudo tee /etc/apt/sources.list.d/bigbluebutton.list
3.Install gpg key
	#curl http://repo.company-group.com/bigbluebutton.org/1-0/gpg.key | sudo apt-key add -
4.Run apt-get update 
	#sudo apt-get update
5.Install bbb1.0, bbb-demo, bbb-chek and bbb-html5
 	#sudo apt-get -y install bigbluebutton bbb-demo bbb-check bbb-html5
6.To enable WebRTC audio
	#sudo bbb-conf --enablewebrtc
7.Complete restart bigbluebutton
	#sudo bbb-conf --clean


Additional Info for debugging problems
- verify that following file have correct ip configured
 		1)/var/www/bigbluebutton/client/conf/config.xml
 		2)/etc/bigbluebutton/nginx/sip.nginx
 		3)/usr/share/red5/webapps/sip/WEB-INF/bigbluebutton-sip.properties
- You can check all the components are working properly using following command
	#sudo bbb-conf —-check
- You can also check more some detailed info using following command
	#sudo bbb-conf —-debug
	

- If you face any ip related problems, or your ip changes, Following commands will change ip in needed files
	#sudo bbb-conf --setip <ip_address_or_hostname>
	#sudo bbb-conf --enablewebrtc
	#sudo bbb-conf --clean
	#sudo bbb-conf --check

- Following are all log files where you can check if there are issues.
	* Nginx
		1)/var/log/nginx/bigbluebutton.access.log 
		2)/var/log/nginx/error.log 
	* Red5
		1)/usr/share/red5/log/deskshare.log
		2)/usr/share/red5/log/sip.log
		3)/usr/share/red5/log/video.log
		4) /usr/share/red5/log/bigbluebutton.log
	* Tomcat7
		1)catalina.yyyy-mm-dd.log	
		2)localhost.yyyy-mm-dd.log	
		3)/var/log/syslog	
		4)/var/log/bigbluebutton
